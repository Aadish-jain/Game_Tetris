# Game_Tetris
Game_Tetris is based on HTML, CSS, and JavaScripts.

In this game of Tetris we will be covering the following inbuilt JavaScript methods:

    addEventListener()
    querySelector()
    querySelectorAll()
    keyCode()
    Math.floor()
    Math.random()
    length
    forEach()
    splice()
    clearInterval()
    setInterval()
    some()
    innerHTML()
    every()
    add()
    remove()
    contains()
